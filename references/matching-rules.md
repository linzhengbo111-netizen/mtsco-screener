# 匹配规则

## 主规则：公司名 + 国家/地区

匹配以公司名称和国家/地区为核心判定依据，其他信号仅作辅助。

## 信号优先级

### 高优先级信号

- 公司名规范化后高度相似
- 国家/地区一致
- 官网域名一致
- 搜索结果中标准公司名与输入公司名非常接近

### 辅助信号

- 联系邮箱域名与官网/公司域名一致
- 产品关键词与进口分析中的产品/HS 线索相关
- 详情页中企业信息与历史客户事实不冲突

## match_status 判定

### confirmed（85–100）

需同时满足：

1. 公司名高度相似
2. 国家/地区一致
3. 至少一项辅助证据支持（官网域名一致 / 详情页企业信息明显匹配）

### likely_match（60–84）

满足：

1. 公司名相似
2. 国家/地区基本一致或部分一致
3. 辅助证据不足

### unconfirmed（0–59）

满足以下任一：

- 搜到结果，但公司名、国家、域名、详情信息之间存在明显冲突
- 证据太弱（仅名称模糊相似、国家不一致、无辅助信号）
- 详情页或进口分析页打开失败

### no_result（0）

满足以下任一：

- 搜索无结果
- 无法进入有效详情页

## 多结果冲突处理

- 输出 top1 候选
- match_confidence 降低（<= 60）
- match_status 降级为 `likely_match` 或 `unconfirmed`
- manual_review_flag = yes
- manual_review_reason = "搜索结果存在多候选，首版仅输出 top1"
- **不中断整批**

## 无 country_region 输入

- 允许执行
- 结果默认降置信度（上限不超过 70）
